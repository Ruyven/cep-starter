# cep-starter

## requirements

**For developing:**

- node.js

**For packaging installers:**

- brew
- makensis

```shell
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew install makensis
```

## install

```shell
git clone git@github.com:adobe-extension-tools/cep-starter.git
cd cep-starter
npm install
```

# develop

```shell
npm start
```

# build & package

```shell
npm run package
```
